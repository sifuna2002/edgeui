Simple Slideshow

A slideshow that allows for HTML content.

Note: This slideshow uses jQuery, and currently only supports one (1) slideshow per page.

Easy To Implement

Copy and paste the HTML/CSS/JS into your project.

Basic HTML Struture

.slider > .item > .item-content > ...

Remember to set the .active class on the slider item you want to show first.

Auto-Generated Navigation

The .slider-nav items will be gernerated based on the number of slide .items there are.
